Troubleshooting

Device not detected?
  • Use a data cable (not just charging)
  • Try a different USB port
  • Put ESP32 in bootloader mode manually
  • Check cable and connections
Manual bootloader mode
  1. Hold BOOT button on ESP32
  2. While holding BOOT, press EN/RST briefly
  3. Release EN/RST, then release BOOT
  4. Try flashing again
Supported hardware
  • ESP32-DevKitC (all variants)
  • ESP32-WROOM-32
  • NodeMCU-32S
  • Most ESP32 development boards
